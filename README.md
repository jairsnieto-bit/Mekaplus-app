# Sistema de Gestión de Guías de Envío - Mekaplus Express

Sistema completo para la generación y administración de guías de envío con generación de PDF masivo.

## 🚀 Características

- ✅ Generación individual de guías
- ✅ Carga masiva desde Excel (.xlsx)
- ✅ Generación de PDF con 3 guías por página
- ✅ Códigos de barras automáticos
- ✅ Panel de administración completo
- ✅ Gestión de usuarios y roles
- ✅ Configuración dinámica del sistema
- ✅ Dashboard con estadísticas

## 📋 Requisitos Previos

- Node.js 18+ 
- PostgreSQL 14+
- npm o yarn
